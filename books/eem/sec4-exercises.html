    4.

    The triangle inequality is the inequality \(\abs{x + y} \le \abs x + \abs y\text{,}\) which is true for any numbers \(x\) and \(y\text{.}\) Confirm the instance of the triangle inequality for the following pairs of numbers \(x\) and \(y\text{.}\) Also place the values \(x\text{,}\) \(y\text{,}\) and \(x+y\) on a number line, and show on your picture the distances that correspond to \(\abs x\text{,}\) \(\abs y\text{,}\) and \(\abs{x+y}\text{.}\)
    • \(x = 2\) and \(y = 5\text{,}\)

    5.

    When we talked about natural numbers, we defined exponentiation as repeated multiplication:
    \begin{equation*} a^n = \underbrace{a \cdot a \cdot \cdots \cdot a}_{n \text{ many}}. \end{equation*}
    The same definition makes sense when \(a\) is any integer, i.e. possibly negative. Note however, that \(n\) must be \(\ge 0\text{.}\) (We can make sense of negative exponents using fractions, but that will come later.) Give a rule for when \(a^n\) is positive versus negative.
    🔗
    Hint.
    Your rule should break into two cases when \(a\) is positive or negative. Handle what happens in each case separately.
    🔗
    🔗
    🔗

    6.

    A square root is the opposite of squaring: \(y\) is a square root of \(x\) means \(y^2 = x\text{.}\) For example, \(2\) is a square root of \(4\) because \(2^2 = 4\text{.}\) Explain why every positive number has exactly two square roots, \(0\) has exactly one square root, and every negative number has no square roots.
    🔗
    🔗

    7.

    With clocks it makes sense to add or subtract—what time will it be after three hours? what time was it three hours ago?—but multiplication doesn’t quite make sense. What would three o’clock times five o’clock even mean? Nonetheless you could try to make sense of it by doing multiplication modulo \(12\text{.}\) Do so to try to make sense of the following>
    • three o’clock times five o’clock,
      🔗
      🔗
    • three o’clock times seven o’clock,
      🔗
      🔗
    • three o’clock times eleven o’clock.
      🔗
      🔗
    Can you give another example where you can do a mathematical calculation but it doesn’t correspond to anything in reality?
    🔗
    🔗

    8.

    Recall that in the integers every number has an additive inverse—a number you add to it to get a sum of \(0\text{.}\) (This was one of the axioms of commutative rings.) This is also true in the integers modulo \(n\text{.}\) Check this in the case of the integers modulo \(12\) by finding the additive inverses for each number. That is, for each \(a \lt 12\) find the number \(b \lt 12\) so that \(a + b \equiv 0 \pmod{12}\text{.}\) Can you give a general rule for what \(-a\text{,}\) the additive inverse of \(a\text{,}\) is in the integers modulo \(12\text{?}\)
    🔗
    🔗

    9.

    A fun fact about the integers modulo a prime \(p\) is that every nonzero number has a multiplicative inverse. That is, for any \(0 \lt a\lt p\) there is \(0 \lt b \lt p\) so that \(ab \equiv 1 \pmod p\text{.}\) Check this fact for the integers modulo \(5\) by finding, for each nonzero number \(\lt 5\) the number that multiply to get \(1 \pmod 5\text{.}\)
    🔗
    🔗

    10.

    Another fun fact about modular arithmetic is that if \(n \gt 1\) is composite then there are zero divisors—nonzero numbers which multiply to have a product of \(0\text{.}\) That is, if \(n\) is composite then there are nonzero \(a,b \lt n\) so that \(ab \equiv 0 \mod n\text{.}\) Check this fact for the integers modulo \(12\)by finding two numbers that multiply to get \(0 \pmod{12}\text{.}\) How many pairs of zero divisors can you find?
