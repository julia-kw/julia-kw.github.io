    Section 6.3 Motivation

    Suppose you are a scientist in the lab and you are measuring the mass of some chemical. something something using fractions is inconvenient. better to use somethnig like decimal. go gram / mg to gram dot gram fraction
    🔗
    This motivates a different notation for writing numbers. But are we needing new kinds of numbers, or is this just a new way to name what we’ve already got? In other words, are there any of these continuous quantities which we can’t represent as a rational number?
    🔗
    An example comes from geometry. Suppose you have an isosceles right triangle, where the two legs each have length \(1\text{.}\) The Pythagorean theorem tells us how to find the length \(c\) of the third side.
    \begin{equation*} c^2 = 1^2 + 1^2 \end{equation*}
    A little work tells us that \(c\) should be the square root of \(2\text{.}\) This is a number which is irrational.
    🔗

    Theorem 6.3.1.

    The number \(\sqrt 2\) is irrational. That is, there are no integers \(p,q\) so that \(\sqrt 2 = \frac pq\text{.}\)
    🔗
    🔗

    Proof.

    How do you show that there are no such integers \(p,q\text{?}\) One way is to assume that there were such \(p,q\) and show that that is impossible. That is the approach we will take. To that end, suppose that \(p\) and \(q\) are integers so that \(\sqrt 2 = \frac pq\text{.}\) As we discussed in the chapter on rational numbers, any fraction can be written in reduced form where there are no common factors between the numerator and denominator. Let’s thereby assume that we’ve already done any canceling out so \(p\) and \(q\) have no common factors.
    🔗
    Now we do a little algebra.
    \begin{align*} \sqrt 2 \amp= \frac pq\\ 2 \amp= \frac{p^2}{q^2}\\ 2q^2 \amp= p^2 \end{align*}
    What this shows is, \(p^2\) is even. This means that \(p\) is also even, because if it were odd then its square would also be odd. In symbols, \(p = 2k\) for some integer \(k\text{.}\) Substitute that into the equation and do a little more algebra.
    \begin{align*} 2q^2 \amp= (2k)^2 \\ 2q^2 \amp= 4k^2\\ q^2 \amp=2k^2 \end{align*}
    This shows us that \(q^2\) is even, so like before \(q\)is also even. But this is impossible! If \(p\) and \(q\) are both even then they have a common factor of \(2\text{,}\) when they were supposed to have no common factor.
    🔗
    We have seen that our assumption that there are integers \(p,q\) so that \(\sqrt 2 = \frac pq\) is impossible, so \(\sqrt2 \) must be irrational.
    🔗
    🔗
    With some number theory work you can generalize this fact. Say that an integer is a perfect square if it is a square of an integer. And an integer is a perfect \(n\)-th power if it is an \(n\)-th power of an integer. Note that the square root of a perfect square is a rational number, because it is a whole number. And similarly for \(n\)-th roots of perfect \(n\)-th powers.
    🔗

    Theorem 6.3.2.

    The number \(\sqrt k\) is irrational if \(k\) is not a perfect square. And \(\sqrt[n] k\)is irrational if \(k\) is not a perfect \(n\)-th power.
    🔗
    🔗
    The lesson is that there are lots and lots of irrational numbers. Indeed, other familiar numbers such as the circle constant \(\pi\) are also irrational. (Though this requires some hefty mathematics to show.) If we want to be able to talk about numbers like these we have to go beyond fractions of integers.
