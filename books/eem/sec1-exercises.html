    Section 1.9 Exercises

    Exercises Exercises

    For these counting exercises, write your answer as a formula, not as a number. For example, write \((2+3)\cdot 4\) instead of \(20\text{.}\)
    🔗

    1.

    A date night special at a restaurant lets you pick either a soup or salad as a starter, then an entree, and finally a dessert. If there are three soups, two salads, five entrees, and four desserts, how many different meals can you choose? If you and your date independently choose your own meals, how many combinations are there for how the two of you can order?
    🔗
    🔗

    2.

    In a race with \(6\) racers, how many different outcomes are there for who gets first through sixth place?
    🔗
    Hint.
    Think of choosing an outcome as first picking who comes in first, then picking a different person for second, and so on to sixth place. How many options do you have for first? Then how many for second? And so on.
    🔗
    🔗
    🔗

    3.

    How many ways are there to pick a \(4\) digit PIN? A \(6\) digit PIN. A \(n\) digit PIN for any positive whole number \(n\text{?}\)
    🔗
    Hint.
    Think of picking each digit as an independent choice. How many options for one digit? Now put it together for multiple digits.
    🔗
    🔗
    🔗
    In more advanced uses of sets, it’s helpful to talk about mappings or correspondances between them. How do you assign each element of the input set an element of the output set?
    🔗

    Definition 1.9.1. Functions.

    Let \(A\) and \(B\) be sets. A function \(f\) from \(A\) to \(B\text{,}\) which we write as \(f \colon A \to B\) is a way of assigning each input \(x \in A\) to an output \(f(x) \in B\text{.}\)
    🔗
    🔗

    4.

    Consider \(A = \{1, 2, 3\}\) and \(B = \{1, 2, 3, 4, 5\}\text{.}\) Give an example of a function \(f \colon A \to B\) by specifying the three outputs \(f(1), f(2), f(3)\text{.}\)
    🔗
    🔗
    We can use functions to give an alternative definition of exponentiation. Namely, \(a^n\) counts the number of functions \(f \colon N \to A\) from a set \(N\) of size \(n\) to a set \(A\) of size \(a\text{.}\)
    🔗

    5.

    Confirm the two definitions of exponentiation are equivalent for a specific case by counting how many functions there are from a set \(N\) with \(4\) elements to a set \(A\) with \(2\) elements.
    🔗
    Hint.
    Think of choosing a function as choosing an output for the first element of \(N\text{,}\) then the next one, and so on until you’ve picked an output for every function.
    🔗
    🔗
    🔗

    6.

    Confirm that this is in general equivalent to the definition by iterated multiplication by explaining why there are
    \begin{equation*} \underbrace{a \cdot a \cdots a}_{n \text{ many}} \end{equation*}
    functions from a set \(N\) of size \(n\) to a set \(A\) of size \(a\text{.}\)
    🔗
    Hint.
    Same hint as before, but now instead of having a specific value for \(n\) and \(a\) you have to think a step more abstractly.
    🔗
    🔗
    🔗
    A function \(f \colon A \to B\) is called one-to-one if different inputs get sent to different outputs. It is called onto if every element of \(B\) is the output \(f(x)\) for some input \(x \in A\text{.}\) If \(f\) is both one-to-one and onto it is called bijective.
    🔗

    7.

    Let \(A = \{1,2,3\}\) and \(B = \{a, b, c\}\text{.}\) Come up with a function \(f \colon A \to B\) which is one-to-one by specifying the outputs \(f(x)\) for every input \(x \in A\text{.}\) Also come up with a function \(g \colon A \to B\) which is onto. Can you come up with a function \(h \colon A \to B\) which is onto but not one-to-one? Either demonstrate such a function or explain why you think it’s impossible.
    🔗
    🔗

    8.

    Consider \(f \colon \Nbb \to \Nbb\) defined as \(f(n) = n+1\text{.}\) Explain why \(f\) is one-to-one but not onto. Also consider \(g \colon \Nbb \to \Nbb\) defined as \(g(n)\) is the whole number you get rounding down from \(\frac{n}{2}\text{.}\) Explain why \(g\) is onto but not one-to-one.
    🔗
    🔗
    One use of functions is to give a way to compare sizes of sets. If \(A\) and \(B\) are sets, then \(\card A \le \card B\) if there is a one-to-one function \(f \colon A \to B\) and \(\card A = \card B\) if there is a bijective function \(f \colon A \to B\text{.}\)
    🔗

    9.

    Demonstrate that your left hand and right hand have the same number of fingers by describing a bijection from the set of fingers on your left hand to the set of fingers on your right hand.
    🔗
    🔗

    10.

    Demonstrate that any two singletons \(\{a\}\) and \(\{b\}\) have the same cardinality by giving a bijective function between them.
