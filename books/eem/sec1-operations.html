    Section 1.4 Operations on sets

    First let’s talk about how to compare sets.
    🔗

    Definition 1.4.1. Subsets and set equality.

    A set \(A\) is a subset of a set \(B\text{,}\) written \(A \subseteq B\text{,}\) if every element of \(A\) is an element of \(B\text{.}\) That is, if \(x \in A\) then \(x \in B\text{.}\) Similar to how we write \(\ne\) for not equals, we write \(A \not \subseteq B\) to say that \(A\) is not a subset of \(B\text{.}\)
    🔗
    Two sets \(A\) and \(B\) are equal if they have the same elements. That is, for any object \(x\text{,}\) we have \(x \in A\) if and only if \(x \in B\text{.}\) Note that this says \(A = B\) if and only if \(A \subseteq B\) and \(B \subseteq A\text{.}\)
    🔗
    🔗

    Checkpoint 1.4.2.

    Explain why the “note that” from the definition is true.
    🔗
    🔗
    The following summarizes some important properties of the subset relation \(\subseteq\text{.}\) Each property is given its formal mathematical name.
    🔗

    Theorem 1.4.3. Properties of \(\subseteq\).

    Reflexivity
    🔗
    \(A \subseteq A\)
    Antisymmetry
    🔗
    If \(A \subseteq B\) and \(B \subseteq A\) then \(A = B\text{.}\)
    Transitivity
    🔗
    If \(A \subseteq B\) and \(B \subseteq C\) then \(A \subseteq C\text{.}\)
    🔗
    🔗
    These symbols can be unpacked into ordinary language. Reflexivity says that any set is contained in itself. Antisymmetry says that if two sets are contained in each other they must be the same. Transitivity says that if a set is contained inside a bigger set contained inside a yet bigger set, then the smallest set is contained inside the biggest.
    🔗

    Checkpoint 1.4.4.

    When comparing numbers, either they are the same or one is bigger than the other. This isn’t the case with sets. Given two sets it is possible neither one is a subset of the other. Demonstrate this by finding two sets \(A\) and \(B\) so that \(A \not \subseteq B\) and \(B \not \subseteq A\text{.}\)
    🔗
    🔗
    In addition to comparing sets, we also want to combine them to make new sets. There are multiple ways we can do this
    🔗

    Definition 1.4.5. Intersection.

    The intersection of two sets is the set of their common elements. We use \(\cap\) as the intersection symbol. Using set-builder notation:
    \begin{equation*} A \cap B = \{ x : x \in A \mand x \in B \}. \end{equation*}
    🔗
    🔗

    Example 1.4.6.

    Consider \(X = \{1, 3, 5, 7, 9\}\) and \(Y = \{2, 3, 5, 7\}\text{.}\) Then
    \begin{equation*} X \cap Y = \{ 3, 5, 7 \}. \end{equation*}
    🔗
    🔗
    Consider two disjoint sets, meaning they have no elements in common. For instance, consider \(A = \{1, 3, 5\}\) and \(B = \{2, 4, 6\}\text{.}\) Then their intersection will have no elements, because nothing is in both \(A\) and \(B\text{,}\) and be the empty set. In symbols, \(A \cap B = \emptyset\text{.}\)
    🔗

    Definition 1.4.7. Union.

    The union of two sets is the set containing all elements from each of the sets. We use \(\cup\) as the union symbol. Using set-builder notation:
    \begin{equation*} A \cup B = \{ x : x \in A \mor x \in B \}. \end{equation*}
    🔗
    🔗

    Example 1.4.8.

    Consider \(X = \{1, 3, 5, 7, 9\}\) and \(Y = \{2, 3, 5, 7\}\text{.}\) Then
    \begin{equation*} X \cup Y = \{ 1, 2, 3, 5, 7, 9 \}. \end{equation*}
    🔗
    🔗

    Definition 1.4.9. Set difference.

    The set difference of \(B\) from \(A\text{,}\) written \(A \setminus B\text{,}\) is the set you obtain by throwing everything from \(B\) out of \(A\text{.}\) Using set-builder notation:
    \begin{equation*} A \setminus B = \{ x \in A : x \not \in B \}. \end{equation*}
    🔗
    🔗

    Example 1.4.10.

    Consider \(X = \{1, 3, 5, 7, 9\}\) and \(Y = \{2, 3, 5, 7\}\text{.}\) Then
    \begin{equation*} X \setminus Y = \{ 1, 9 \} \end{equation*}
    and
    \begin{equation*} Y \setminus X = \{ 2 \}. \end{equation*}
    🔗
    🔗
    Often you want to express the idea of “first this thing, then that thing”. For example, when you buy a ticket at a movie theater you might have to pick the row and then the seat in that row. In mathematics, we abstract this kind of choice in the object of an ordered pair. Namely, we write \((x,y)\) for the ordered pair whose first object is \(x\) and whose second object is \(y\text{.}\)
    🔗

    Definition 1.4.11. Cartesian product.

    Let \(A\) and \(B\) be sets. The Cartesian product of \(A\) and \(B\text{,}\) written \(A \times B\text{,}\) is the set of all ordered pairs \((x,y)\) where \(x \in A\) and \(y \in B\text{.}\) In set-builder notation:
    \begin{equation*} A \times B = \{ (x,y) : x \in A \mand y \in B \}. \end{equation*}
    🔗
    🔗

    Checkpoint 1.4.12.

    What is the Cartesian product of \(A = \{1, 2\}\) and \(B = \{3, 4\}\text{?}\)
    🔗
    Answer.
    \(A \times B = \{ (1,3), (1,4), (2,3), (2,4) \}\)
    🔗
    🔗
    🔗
    You can visualize a Cartesian product as a rectangle. The vertical side represents the set \(A\) with all of its elements and the horizontal side represents the set \(B\) with all of its elements. To specify a position inside the rectangle you need to give both the vertical and horizontal position, which amounts to specifying first an element from \(A\) and second an element from \(B\text{.}\) Thus, the positions inside correspond to all the possible ordered pairs from \(A\) followed by \(B\text{.}\)
